# AzureAIAvatarBlazor.MAFFoundry

## Overview

Class library for integrating with **Microsoft Agents Framework (MAF) Foundry** - the cloud-based agent orchestration platform.

## Purpose

This library provides:

- Cloud-based agent capabilities using Azure AI services
- Integration with MAF Foundry for agent management
- Azure AI-powered agent orchestration
- Scalable agent execution in the cloud
